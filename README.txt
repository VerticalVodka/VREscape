Keys to use Triality without additional Hardware:

Numeric Buttons 1-4 -> Buttons in game

A -> Radio Rotary Left
D -> Radio Rotary Right


Left Arrow -> Safe Rotary Left
Right Arrow -> Safe Rotary Right

L -> Skip Level

K -> Button 5 (=Hidden Button in Drawer)

C -> Close Safe (Only useful when Hardware is attached)
S -> Unlock Safe

D -> Unlock Drawer
